# ytv2md - Youtube Video to Markdown

A greasemonkey/tampermonkey/violentmonkey script that will generate a markdown of a youtube video including various metadata information, description, user notes, transcription, etc. Can use a local server(not included) to read, update, and write files to specific locations. Can be used with apps like Obsidian.
